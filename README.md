# Kh2anywhere

## Installation

```bash
$ bin/setup
```

## Usage

Output your kindle highlights to `mds` directory.

```bash
$ ./all.sh -m <YOUR_KINDLE_EMAIL> -p <YOUR_KINDLE_PASSWORD>
```

Or

```bash
$ ./all.sh -m <YOUR_KINDLE_EMAIL>

Input your kindle password?: <INPUT_YOUR_KINDLE_PASSWORD>
```
